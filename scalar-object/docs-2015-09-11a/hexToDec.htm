hexToDec

Returns the decimal equivalent of the hexadecimal number represented by the hex_string argument. hexToDec() converts a hexadecimal string to a decimal number.

hexToDec() will ignore any non-hexadecimal characters it encounters.

Parameters

hex_string
The hexadecimal string to convert

Return Values

The decimal representation of hex_string

Example #1 hexToDec() example

<?php
var_dump(hexToDec("See"));
var_dump(hexToDec("ee"));
// both print "int(238)"

var_dump(hexToDec("that")); // print "int(10)"
var_dump(hexToDec("a0")); // print "int(160)"
?>
Note: The function can convert numbers that are too large to fit into the platforms integer type, larger values are returned as float in that case.
