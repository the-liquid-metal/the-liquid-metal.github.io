compare

compare( string $str2 ): int

Context: string.
Alias of strcmp.
Binary safe comparison.

Note that this comparison is case sensitive.

Parameters

$str2
The second string.

Return Values

  • < 0 if $scalarObject->val is less than $str2.
  • > 0 if $scalarObject->val is greater than $str2.
  • and 0 if they are equal.
