firstNthCompare

firstNthCompare( string $str2, int $length ): int

Context: string.
Alias of strncmp.
Binary safe comparison of the first n characters.

This function is similar to compare(), with the difference that you can specify the (upper limit of the) number of characters from each string to be used in the comparison.

Note that this comparison is case sensitive.

Parameters

$str2
The second string.
$length
Number of characters to use in the comparison.

Return Values

  • < 0 if $scalarObject->val is less than $str2.
  • > 0 if $scalarObject->val is greater than $str2.
  • and 0 if they are equal.
