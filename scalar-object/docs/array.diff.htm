diff( array $array2, array ... ): array

Context: array.
Alias of array_diff.
Computes the difference.

diff

Compares $scalarObject->val against one or more other arrays and returns the values in $scalarObject->val that are not present in any of the other arrays.

Parameters

$array2
An array to compare against
...
More arrays to compare against

Return Values

An array containing all the entries from $scalarObject->val that are not present in any of the other arrays.

Example #1 diff() example

       "green", "red", "blue", "red");
      $array2 = array("b" => "green", "yellow", "red");
      $result = diff($array1, $array2);
      
      print_r($result);
      ?>
Multiple occurrences in $array1 are all treated the same way. This will output : 
      Array
      (
          [1] => blue
      )
Note: Two elements are considered equal if and only if (string) $elem1 === (string) $elem2. In words: when the string representation is the same.
Note: This function only checks one dimension of a n-dimensional array. Of course you can check deeper dimensions by using diff($array1[0], $array2[0]);.
