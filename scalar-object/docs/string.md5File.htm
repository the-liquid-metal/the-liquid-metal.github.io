md5File( bool $rawOutput = false ): string

Context: string.
Alias of md5_file.
Calculate the md5 hash of a given file.

md5File

Calculates the MD5 hash of the file specified by the $scalarObject->val parameter using the » RSA Data Security, Inc. MD5 Message-Digest Algorithm, and returns that hash. The hash is a 32-character hexadecimal number.

Parameters

$rawOutput
When TRUE, returns the digest in raw binary format with a length of 16.

Return Values

Returns a string on success, FALSE otherwise.
