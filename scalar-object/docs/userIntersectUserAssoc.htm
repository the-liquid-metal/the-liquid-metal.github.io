Context: array.
Alias of array_uintersect_uassoc.
Computes the intersection with additional index check, compares data and indexes using separate callback functions.

userIntersectUserAssoc

Computes the intersection of arrays with additional index check, compares data and indexes by separate callback functions.

Parameters

$array2
The second array.
...
More arrays to compare against
$valueComparator
The comparison function must return an integer less than, equal to, or greater than zero if the first argument is considered to be respectively less than, equal to, or greater than the second.
$keyComparator
Key comparison callback function.

Return Values

An array containing all the values of $scalarObject->val that are present in all the arguments.
