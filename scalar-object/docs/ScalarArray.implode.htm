implode

implode (string $glue = ""): string

Join elements with a string.

see implode.

Join array elements with a $glue string.

Parameters

$glue
Defaults to an empty string.

Return Values

A string containing a string representation of all the internal array elements in the same order, with the glue string between each element. 

        $array = ["lastname", "email", "phone"];
        implode(",", $array); // output: "lastname,email,phone"

        // Empty string when using an empty array:
        implode("hello", []); // output: ""
Note: This function is binary-safe.
