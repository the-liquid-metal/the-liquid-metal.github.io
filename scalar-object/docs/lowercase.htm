lowercase

Description

Returns string with all alphabetic characters converted to lowercase.

Note that 'alphabetic' is determined by the current locale. This means that e.g. in the default "C" locale, characters such as umlaut-A (Ã„) will not be converted.

Parameters

string

The input string.

Return Values

Returns the lowercased string.

Examples

Example #1 lowercase() example

<?php
$str = "Mary Had A Little Lamb and She LOVED It So";
$str = lowercase($str);
echo $str; // Prints mary had a little lamb and she loved it so
?>

Notes

Note: This function is binary-safe.
