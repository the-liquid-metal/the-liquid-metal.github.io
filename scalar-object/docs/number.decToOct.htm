decToOct( ): string

Context: number.
Alias of decoct.
Decimal to octal.

decToOct

Returns a string containing an octal representation of the given $scalarObject->val argument. The largest number that can be converted is 4294967295 in decimal resulting to "37777777777".

Return Values

Octal string representation of $scalarObject->val.
