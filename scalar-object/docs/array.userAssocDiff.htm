userAssocDiff

userAssocDiff( array $array2, array ..., callable $keyComparator ): array

Context: array.
Alias of array_diff_uassoc.
Computes the difference with additional index check which is performed using a callback function.

Compares $scalarObject->val against $array2 and returns the difference. Unlike diff() the array keys are used in the comparison.

Unlike assocDiff() an user supplied callback function is used for the indices comparison, not internal function.

Parameters

$array2
An array to compare against
...
More arrays to compare against
$keyComparator
The comparison function must return an integer less than, equal to, or greater than zero if the first argument is considered to be respectively less than, equal to, or greater than the second.

Return Values

Returns an array containing all the entries from $scalarObject->val that are not present in any of the other arrays.

The "a" => "green" pair is present in both arrays and thus it is not in the output from the function. Unlike this, the pair 0 => "red" is in the output because in the second argument "red" has key which is 1.

The equality of 2 indices is checked by the user supplied callback function.
Note: This function only checks one dimension of a n-dimensional array. Of course you can check deeper dimensions by using, for example, userAssocDiff($array1[0], $array2[0], "key_compare_func");.
