changeKeyCase

Returns an array with all keys from array lowercased or uppercased. Numbered indices are left as is.

Parameters

array
The array to work on
case
Either CASE_UPPER or CASE_LOWER (default)

Return Values

Returns an array with its keys lower or uppercased, or FALSE if array is not an array.

Errors/Exceptions

Throws E_WARNING if array is not an array.

Example #1 changeKeyCase() example

       1, "SecOnd" => 4);
      print_r(changeKeyCase($input_array, CASE_UPPER));
      ?>
The above example will output:
      Array
      (
          [FIRST] => 1
          [SECOND] => 4
      )
Note: If an array has indices that will be the same once run through this function (e.g. "keY" and "kEY"), the value that is later in the array will override other indices.
