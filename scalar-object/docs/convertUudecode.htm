convertUudecode

convertUudecode() decodes a uuencoded string.

Parameters

data
The uuencoded data.

Return Values

Returns the decoded data as a string or FALSE on failure.

Example #1 convertUudecode() example

<?php
/* Can you imagine what this will print? :) */
echo convertUudecode("+22!L;W9E(%!(4\"$`\n`");
?>
