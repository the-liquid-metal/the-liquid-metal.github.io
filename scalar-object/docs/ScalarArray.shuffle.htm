shuffle

shuffle (void): static

randomizes the order of the elements in an array.

see shuffle.

This function shuffles (randomizes the order of the elements in) an internal array.

Return Values

TRUE on success or FALSE on failure.

        $numbers = range(1, 20);
        shuffle($numbers);
        foreach ($numbers as $number) {
            echo "$number ";
        }
Note:This function assigns new keys to the elements in internal array. It will remove any existing keys that may have been assigned, rather than just reordering the keys.
