sum

sum() returns the sum of values in an array.

Parameters

array

The input array.

Return Values

Returns the sum of values as an integer or float.

Example #1 sum() examples

<?php
$a = array(2, 4, 6, 8);
echo "sum(a) = " . sum($a) . "\n";

$b = array("a" => 1.2, "b" => 2.3, "c" => 3.4);
echo "sum(b) = " . sum($b) . "\n";
?>

The above example will output:

sum(a) = 20
sum(b) = 6.9
