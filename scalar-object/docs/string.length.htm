length

length( ): int

Context: string.
Alias of strlen.
Get string length.

Returns the length of the given $scalarObject->val.

Return Values

The length of the $scalarObject->val on success, and 0 if the $scalarObject->val is empty.

Example #1 A length() example
Note: length() returns the number of bytes rather than the number of characters in a string.
Note: length() returns NULL when executed on arrays, and an E_WARNING level error is emitted.
