fillKeys

fillKeys (mixed $value): self

Fills with values, specifying keys.

see array_fill_keys.

Fills an array with the value of the $value parameter, using the values of the internal array array as keys.

Parameters

$value
Value to use for filling

Return Values

Returns the filled array

        $keys = array["foo", 5, 10, "bar"];
        fillKeys($keys, "banana"); // output: ["foo" => "banana", 5 => "banana", 10 => "banana", "bar" => "banana"]
