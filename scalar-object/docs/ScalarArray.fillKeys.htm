fillKeys

fillKeys( mixed $value ): array

Alias of array_fill_keys.
Fills with values, specifying keys.

Fills an array with the value of the $value parameter, using the values of the $scalarObject->val array as keys.

Parameters

$value
Value to use for filling

Return Values

Returns the filled array

The above example will output:
