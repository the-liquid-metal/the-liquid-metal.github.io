lowercaseFirst( ): string

Context: string.
Alias of lcfirst.
Make first character lowercase.

lowercaseFirst

Returns a string with the first character of $scalarObject->val, lowercased if that character is alphabetic.

Note that 'alphabetic' is determined by the current locale. For instance, in the default "C" locale characters such as umlaut-a (Ã¤) will not be converted.

The resulting string.
