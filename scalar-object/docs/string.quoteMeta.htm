quoteMeta

quoteMeta( ): string

Context: string.
Alias of quotemeta.
Quote meta characters.

Returns a version of str with a backslash character (\) before every character that is among these: . \ + * ? [ ^ ] ( $ )

Return Values

The string with meta characters quoted, or FALSE if an empty string is given as $scalarObject->val.

Note:This function is binary-safe.
