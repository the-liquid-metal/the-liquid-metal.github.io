md5

md5( bool $rawOutput = false )

Context: string.
Alias of md5.
Calculate the md5 hash.

Calculates the MD5 hash of $scalarObject->val using the » RSA Data Security, Inc. MD5 Message-Digest Algorithm, and returns that hash.

Parameters

$rawOutput
If it is set to TRUE, then the md5 digest is instead returned in raw binary format with a length of 16.

Return Values

Returns the hash as a 32-character hexadecimal number.

Example #1 A md5() example
Note: Secure password hashing
It is not recommended to use this function to secure passwords, due to the fast nature of this hashing algorithm. For details see fasthash.
