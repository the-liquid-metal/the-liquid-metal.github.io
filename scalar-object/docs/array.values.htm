values

values( ): array

Context: array.
Alias of array_values.
Returns all values.

values() returns all the values from the $scalarObject->val and indexes the array numerically.

Return Values

An indexed array of values.

Example #1 values() example

       "XL", "color" => "gold");
      print_r(values($array));
