firstOccurSubstr( mixed $needle, bool $beforeNeedle = false ): string

Context: string.
Alias of strstr.
Find the first occurrence of a substring.

firstOccurSubstr

Returns part of $scalarObject->val string starting from and including the first occurrence of $needle to the end of $scalarObject->val.

Note: This function is case-sensitive. For case-insensitive searches, use firstOccurInsenSubstr().
Note: If you only want to determine if a particular $needle occurs within $scalarObject->val, use the faster and less memory intensive function firstOccurPosition() instead.

Parameters

$needle
If $needle is not a string, it is converted to an integer and applied as the ordinal value of a character.
$beforeNeedle
If TRUE, firstOccurSubstr() returns the part of the $scalarObject->val before the first occurrence of the $needle (excluding the needle).

Return Values

The portion of string, or FALSE if $needle is not found.
