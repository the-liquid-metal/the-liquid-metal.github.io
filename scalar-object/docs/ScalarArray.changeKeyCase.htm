changeKeys

changeKeys (int $type = CASE_LOWER): static

Changes the case of all keys.

see array_change_key_case.

Returns an array with all keys from internal array lowercased or uppercased. Numbered indices are left as is.

Parameters

$type
Either CASE_UPPER, CASE_LOWER, CASE_UPPER_FIRST, CASE_UPPER_WORDS, or INT_INDEX

Return Values

An array with its keys lowered or uppercased stored in internal array.

Note: If an array has indices that will be the same once run through this function (e.g. "keY" and "kEY"), the value that is later in the array will override other indices.

values

values() returns all the values from the internal array and indexes the array numerically.

Return Values

An indexed array of values.

        $sa = new ScalarArray("float", true, ["FirSt" => 1, "SecOnd" => 2.3, "THIRD" => null, "fOURTH aND fIFTH" => NAN]);
        $sa->changeKeys(CASE_LOWER);       // output: ["first" => 1, "second" => 2.3, "third" => null, "fourth and fifth" => NAN]
        $sa->changeKeys(CASE_UPPER);       // output: ["FIRST" => 1, "SECOND" => 2.3, "THIRD" => null, "FOURTH AND FIFTH" => NAN]
        $sa->changeKeys(CASE_UPPER_FIRST); // output: ["First" => 1, "Second" => 2.3, "Third" => null, "Fourth and fifth" => NAN]
        $sa->changeKeys(CASE_UPPER_WORDS); // output: ["First" => 1, "Second" => 2.3, "Third" => null, "Fourth And Fifth" => NAN]
        $sa->changeKeys(INT_INDEX);        // output: [      0 => 1,        1 => 2.3,       3 => null,                  4 => NAN]
