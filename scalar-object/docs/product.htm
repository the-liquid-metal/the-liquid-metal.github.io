product

product() returns the product of values in an array.

Parameters

array
The array.

Return Values

Returns the product as an integer or float.

Example #1 product() examples

<?php

$a = array(2, 4, 6, 8);
echo "product(a) = " . product($a) . "\n";
echo "product(array()) = " . product(array()) . "\n";

?>
The above example will output:
product(a) = 384
product(array()) = 1
