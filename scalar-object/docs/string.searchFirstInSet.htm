searchFirstInSet

searchFirstInSet( string $charList ): string

Context: string.
Alias of strpbrk.
Search a string for any of a set of characters.

searchFirstInSet() searches the $scalarObject->val string for a $charList.

Parameters

$charList
This parameter is case sensitive.

Return Values

A string starting from the character found, or FALSE if it is not found.
