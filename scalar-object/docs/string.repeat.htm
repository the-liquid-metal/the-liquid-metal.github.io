repeat

repeat( int $multiplier ): string

Context: string.
Alias of str_repeat.
Repeat a string.

Returns $scalarObject->val repeated $multiplier times.

Parameters

input
The string to be repeated.
multiplier

Number of time the $scalarObject->val string should be repeated.

$multiplier has to be greater than or equal to 0. If the $multiplier is set to 0, the function will return an empty string.

Return Values

Returns the repeated string.
