intDivision( int $divisor ): int

Context: number.
Alias of intdiv.
Integer division.

intDivision

Parameters

$divisor
Number which divides the $scalarObject->val.

Return Values

The integer division of $scalarObject->val by $divisor.

Errors/Exceptions

If $divisor is 0, a DivisionByZeroError exception is thrown. If the $scalarObject->val is PHP_INT_MIN and the $divisor is -1, then an ArithmeticError exception is thrown.
