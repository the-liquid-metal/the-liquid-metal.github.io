intDivision

Returns the integer division of numerator by divisor.

Parameters

numerator

Number to be divided.

divisor

Number which divides the numerator.

Return Values

The integer division of numerator by divisor.

Errors/Exceptions

If divisor is 0, a DivisionByZeroError exception is thrown. If the numerator is PHP_INT_MIN and the divisor is -1, then an ArithmeticError exception is thrown.

Example #1 intDivision() example

<?php
var_dump(intDivision(3, 2));
var_dump(intDivision(-3, 2));
var_dump(intDivision(3, -2));
var_dump(intDivision(-3, -2));
var_dump(intDivision(PHP_INT_MAX, PHP_INT_MAX));
var_dump(intDivision(PHP_INT_MIN, PHP_INT_MIN));
var_dump(intDivision(PHP_INT_MIN, -1));
var_dump(intDivision(1, 0));
?>
