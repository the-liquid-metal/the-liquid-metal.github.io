countValue

countValue (void): self

Counts all the values.

see array_count_values.

countValue() returns an array using the values of internal array as keys and their frequency in internal array as values.

Return Values

An associative array of values from internal array as keys and their count as value.

Errors/Exceptions

Throws E_WARNING for every element which is not string or integer.

        $arr = new ScalarArray("string", false, ["1", "hello", "1", "world", "hello"]);
        $arr->countValue(); // output: ["1" => 2, "hello" => 2, "world" => 1]
