hexToBin

hexToBin( ): string

Context: string.
Alias of hex2bin.
Convert hexadecimal string into binary string.

Decodes a hexadecimally encoded binary string.

Caution
This function does NOT convert a hexadecimal number to a binary number. This can be done using the base_convert() function.

Return Values

Returns the binary representation of $scalarObject->val or FALSE on failure.

Errors/Exceptions

If the $scalarObject->val is of odd length or invalid hexadecimal string an E_WARNING level error is thrown.
