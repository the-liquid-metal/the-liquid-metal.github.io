expm1

expm1( )

Context: number
Alias of expm1
Returns exp(number) - 1, computed in a way that is accurate even when the value of number is close to zero

expm1() returns the equivalent to 'exp($scalarObject->val) - 1' computed in a way that is accurate even if the value of $scalarObject->val is near zero, a case where 'exp ($scalarObject->val) - 1' would be inaccurate due to subtraction of two numbers that are nearly equal.

Return Values

'e' to the power of $scalarObject->val minus one
