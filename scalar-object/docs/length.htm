length

Returns the length of the given string.

Parameters

string

The string being measured for length.

Return Values

The length of the string on success, and 0 if the string is empty.

Example #1 A length() example

<?php
$str = 'abcdef';
echo length($str); // 6

$str = ' ab cd ';
echo length($str); // 7
?>

Note:

length() returns the number of bytes rather than the number of characters in a string.

Note:

length() returns NULL when executed on arrays, and an E_WARNING level error is emitted.
