lowercase

lowercase( ): string

Context: string.
Alias of strtolower.
Make a lowercase.

Returns $scalarObject->val with all alphabetic characters converted to lowercase.

Note that 'alphabetic' is determined by the current locale. This means that e.g. in the default "C" locale, characters such as umlaut-A (Ã„) will not be converted.

Return Values

The lowercased string.
