uppercase

uppercase( ): string

Context: string.
Alias of strtoupper.
Make uppercase.

Returns $scalarObject->val with all alphabetic characters converted to uppercase.

Note that 'alphabetic' is determined by the current locale. For instance, in the default "C" locale characters such as umlaut-a (Ã¤) will not be converted.

Return Values

The uppercased string.
