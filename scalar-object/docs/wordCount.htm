wordCount( int $format = 0, string $charList = null ): mixed

Context: string.
Alias of str_word_count.
Return information about words used in a string.

wordCount

Counts the number of words inside $scalarObject->val. If the optional $format is not specified, then the return value will be an integer representing the number of words found. In the event the $format is specified, the return value will be an array, content of which is dependent on the $format. The possible value for the $format and the resultant outputs are listed below.

For the purpose of this function, 'word' is defined as a locale dependent string containing alphabetic characters, which also may contain, but not start with "'" and "-" characters.

Parameters

$format

Specify the return value of this function. The current supported values are:

  • 0 - returns the number of words found
  • 1 - returns an array containing all the words found inside the $scalarObject->val
  • 2 - returns an associative array, where the key is the numeric position of the word inside the $scalarObject->val and the value is the actual word itself
$charlist
A list of additional characters which will be considered as 'word'

Return Values

An array or an integer, depending on the $format chosen.
