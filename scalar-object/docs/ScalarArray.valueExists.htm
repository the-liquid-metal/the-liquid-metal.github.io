valueExists

valueExists (mixed $needle, bool $strict = false): bool

Checks if a value exists.

see in_array.

Searches internal array for $needle using loose comparison unless $strict is set.

Parameters

$needle
The searched value. If $needle is a string, the comparison is done in a case-sensitive manner.
$strict
If set to true then it will also check the types of the $needle in the internal array.

Return Values

Returns true if $needle is found in the array, false otherwise.

        $a = new ScalarArray("string", false, ["Mac", "NT", "Irix", "Linux"]);
        if ($a->valueExists("Irix")) echo "Got Irix";
        if ($a->valueExists("mac")) echo "Got mac";
        // output:
        //> Got Irix

        $a = new ScalarArray("float", false, [1.23, 2.34, 3.45, 4.56]);
        if ($a->valueExists("2.34", true)) echo "'2.34' found with strict check\n";
        if ($a->valueExists(3.45, true)) echo "3.45 found with strict check\n";
        // output:
        //> 3.45 found with strict check

        $a = new ScalarArray("array", false, [["a", "b"], ["c", "d"], ["e", "f"]];
        if ($a->valueExists(["a", "b"])) echo "'ab' was found\n";
        if ($a->valueExists(["a", "c"])) echo "'ac' was found\n";
        // output:
        //> 'ab' was found
