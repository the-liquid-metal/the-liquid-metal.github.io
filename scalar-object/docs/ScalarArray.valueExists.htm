inArray

inArray( mixed $needle, bool $strict = FALSE ): bool

Alias of in_array.
Checks if a value exists.

Searches $scalarObject->val for $needle using loose comparison unless $strict is set.

Parameters

$needle

The searched value.

Note: If $needle is a string, the comparison is done in a case-sensitive manner.
$strict
If it is set to TRUE then the inArray() function will also check the types of the $needle in the $scalarObject->val.

Return Values

Returns TRUE if $needle is found in the array, FALSE otherwise.

Example #1 inArray() example

            $os = ["Mac", "NT", "Irix", "Linux"];
            if (inArray("Irix", $os)) {
                echo "Got Irix";
            }
            if (inArray("mac", $os)) {
                echo "Got mac";
            }
The second condition fails because inArray() is case-sensitive, so the program above will display: 
Got Irix

Example #2 inArray() with strict example

            $a = ["1.10", 12.4, 1.13];

            if (inArray("12.4", $a, true)) {
                echo "'12.4' found with strict check\n";
            }

            if (inArray(1.13, $a, true)) {
                echo "1.13 found with strict check\n";
            }
The above example will output:
1.13 found with strict check

Example #3 inArray() with an array as needle

            $a = [
                ["p", "h"],
                ["p", "r"],
                "o"
            ];

            if (inArray(["p", "h"], $a)) {
                echo "'ph' was found\n";
            }

            if (inArray(["f", "i"], $a)) {
                echo "'fi' was found\n";
            }

            if (inArray("o", $a)) {
                echo "'o' was found\n";
            }
The above example will output:
            'ph' was found
            'o' was found
