nlToBr

nlToBr( boolean $isXhtml = true ): string

Context: string.
Alias of nl2br.
Inserts HTML line breaks before all newlines.

Returns $scalarObject->val with '<br />' or '<br>' inserted before all newlines (\r\n, \n\r, \n and \r).

Parameters

$isXhtml
Whether to use XHTML compatible line breaks or not.

Return Values

The altered string.

The above example will output:
      foo isn't<br />
