flip

flip (void): self

Exchanges all keys with their associated values.

see array_flip.

flip() returns an array in flip order, i.e. keys from internal array become values and values from internal array become keys.

Note that the values of internal array need to be valid keys, i.e. they need to be either integer or string. A warning will be emitted if a value has the wrong type, and the key/value pair in question will not be included in the result.

If a value has several occurrences, the latest key will be used as its value, and all others will be lost.

Return Values

The flipped array on success and NULL on failure.

            $input = ["oranges", "apples", "pears"];
            flip($input); // output: ["oranges" => 0, "apples" => 1, "pears" => 2]

            $input = ["a" => 1, "b" => 1, "c" => 2];
            flip($input); // output: [1 => "b", 2 => "c"]
