searchFirstInSet

searchFirstInSet() searches the haystack string for a char_list.

Parameters

haystack

The string where char_list is looked for.

char_list

This parameter is case sensitive.

Return Values

Returns a string starting from the character found, or FALSE if it is not found.

Example #1 searchFirstInSet() example

<?php

$text = 'This is a Simple text.';

// this echoes "is is a Simple text." because 'i' is matched first
echo searchFirstInSet($text, 'mi');

// this echoes "Simple text." because chars are case sensitive
echo searchFirstInSet($text, 'S');
?>
