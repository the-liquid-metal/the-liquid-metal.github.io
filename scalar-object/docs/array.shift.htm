shift( ): array

Context: array.
Alias of array_shift.
Shifts an element off the beginning.

shift

shift() shifts the first value of the $scalarObject->val off and returns it, shortening the $scalarObject->val by one element and moving everything down. All numerical array keys will be modified to start counting from zero while literal keys won't be touched.

Return Values

The shifted value, or NULL if $scalarObject->val is empty or is not an array.
