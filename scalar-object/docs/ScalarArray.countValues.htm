countValues

countValues( ): array

Alias of array_count_values.
Counts all the values.

countValues() returns an array using the values of $scalarObject->val as keys and their frequency in $scalarObject->val as values.

Return Values

An associative array of values from $scalarObject->val as keys and their count as value.

Errors/Exceptions

Throws E_WARNING for every element which is not string or integer.

Example #1 countValues() example

            $array = [1, "hello", 1, "world", "hello"];
            print_r(countValues($array));
The above example will output:
            Array
            (
                [1] => 2
                [hello] => 2
                [world] => 1
            )
