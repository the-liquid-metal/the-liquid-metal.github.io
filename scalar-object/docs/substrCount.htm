substrCount( string $needle, int $offset, int $length ): int

Context: string.
Alias of substr_count.
Count the number of substring occurrences.

substrCount

substrCount() returns the number of times the $needle substring occurs in the $scalarObject->val string. Please note that $needle is case sensitive.

Note: This function doesn't count overlapped substrings. See the example below!

Parameters

$needle
The substring to search for
$offset
The offset where to start counting
$length
The maximum length after the specified offset to search for the substring. It outputs a warning if the offset plus the length is greater than the $scalarObject->val length.

Return Values

This function returns an integer.

Example #1 A substrCount() example

       14
      echo substrCount($text, 'is', 5, 10);

      // prints only 1, because it doesn't count overlapped substrings
      $text2 = 'gcdgcdgcd';
      echo substrCount($text2, 'gcdgcd');
