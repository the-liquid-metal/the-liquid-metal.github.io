isNan( ): bool

Context: number.
Alias of is_nan.
Finds whether a value is not a number.

isNan

Checks whether $scalarObject->val is 'not a number', like the result of acos(1.01).

Return Values

TRUE if $scalarObject->val is 'not a number', else FALSE.
