hexToDec( ): number

Context: string.
Alias of hexdec.
Hexadecimal to decimal.

hexToDec

Returns the decimal equivalent of the hexadecimal number represented by the $scalarObject->val argument. hexToDec() converts a hexadecimal string to a decimal number.

hexToDec() will ignore any non-hexadecimal characters it encounters.

Return Values

The decimal representation of $scalarObject->val

Note: The function can convert numbers that are too large to fit into the platforms integer type, larger values are returned as float in that case.
