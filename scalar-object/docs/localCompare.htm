localCompare

Note that this comparison is case sensitive, and unlike compare() this function is not binary safe.

localCompare() uses the current locale for doing the comparisons. If the current locale is C or POSIX, this function is equivalent to compare().

Parameters

str1
The first string.
str2
The second string.

Return Values

  • < 0 if $scalarObject->val is less than $str2.
  • > 0 if $scalarObject->val is greater than $str2.
  • and 0 if they are equal.
