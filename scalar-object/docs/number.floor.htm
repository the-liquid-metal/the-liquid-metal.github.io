floor( ): mixed

Context: number.
Alias of floor.
Round fractions down.

floor

Returns the next lowest integer value (as float) by rounding down $scalarObject->val if necessary.

Return Values

$scalarObject->val rounded to the next lowest integer. The return value of floor() is still of type float because the value range of float is usually bigger than that of integer. This function returns FALSE in case of an error (e.g. passing an array).
