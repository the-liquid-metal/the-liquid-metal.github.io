Context: string.
Alias of convert_cyr_string.
Convert from one Cyrillic character set to another.

convertCyrStr

Converts from one Cyrillic character set to another.

Parameters

$from
The source Cyrillic character set, as a single character.
$to
The target Cyrillic character set, as a single character.

Supported characters are:

  • k - koi8-r
  • w - windows-1251
  • i - iso8859-5
  • a - x-cp866
  • d - x-cp866
  • m - x-mac-cyrillic

Return Values

The converted string of $scalarObject->val.

Note: This function is binary-safe.
