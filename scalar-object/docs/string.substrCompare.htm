substrCompare

substrCompare( string $str, int $offset, int $length = null, boolean $caseInsensitivity = false ): int

Context: string.
Alias of substr_compare.
Binary safe comparison of two strings from an offset, up to length characters.

substrCompare() compares $scalarObject->val from position $offset with $str up to $length characters.

Parameters

$str
The secondary string being compared.
$offset
The start position for the comparison. If negative, it starts counting from the end of the string.
$length
The length of the comparison. The default value is the largest of the length of the $str compared to the length of $scalarObject->val less the $offset.
$caseInsensitivity
TRUE if comparison is case insensitive, otherwise FALSE.

Return Values

  • < 0 if $scalarObject->val from position $offset is less than $str.
  • > 0 if it is greater than $str.
  • 0 if they are equal.
  • FALSE and prints a warning, if $offset >= the length of $scalarObject->val, or the $length is set and is less than 1 (prior to PHP 5.6).
