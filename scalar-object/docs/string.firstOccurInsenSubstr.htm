firstOccurInsenSubstr( mixed $needle, bool $beforeNeedle = false ): string

Context: string.
Alias of stristr.
Find the first occurrence of a case-insensitive substring.

firstOccurInsenSubstr

Returns all of $scalarObject->val starting from and including the first occurrence of $needle to the end.

Parameters

$needle
If $needle is not a string, it is converted to an integer and applied as the ordinal value of a character.
$beforeNeedle
If TRUE, firstOccurInsenSubstr() returns the part of the $scalarObject->val before the first occurrence of the $needle (excluding needle).

$needle and $scalarObject->val are examined in a case-insensitive manner.

Return Values

The matched substring. If $needle is not found, returns FALSE.
