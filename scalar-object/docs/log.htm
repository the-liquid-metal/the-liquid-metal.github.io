log( float $base = M_E ): float

context: number.
Alias of log.
Natural logarithm.

log

If the optional $base parameter is specified, log() returns logbase $scalarObject->val, otherwise log() returns the natural logarithm of $scalarObject->val.

Parameters

$base
The optional logarithmic base to use (defaults to 'e' and so to the natural logarithm).

Return Values

The logarithm of $scalarObject->val to $base, if given, or the natural logarithm.
