random

random( int $num = 1 ): mixed

Alias of array_rand.
Picks one or more random entries.

Picks one or more random entries out of $scalarObject->val, and returns the key (or keys) of the random entries.

Parameters

$num
Specifies how many entries should be picked.

Return Values

When picking only one entry, random() returns the key for a random entry. Otherwise, an array of keys for the random entries is returned. This is done so that random keys can be picked from $scalarObject->val as well as random values. Trying to pick more elements than there are in $scalarObject->val will result in an E_WARNING level error, and NULL will be returned.
