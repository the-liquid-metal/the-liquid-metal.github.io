random

random (int $num = 1): static

Alias of array_rand.
Picks one or more random entries.

Picks one or more random entries out of internal array, and returns the key (or keys) of the random entries.

Parameters

$num
Specifies how many entries should be picked.

Return Values

When picking only one entry, random() returns the key for a random entry. Otherwise, an array of keys for the random entries is returned. This is done so that random keys can be picked from internal array as well as random values. Trying to pick more elements than there are in internal array will result in an E_WARNING level error, and NULL will be returned. 

        $sa = new ScalarArray("string",  false, ["Neo", "Morpheus", "Trinity", "Cypher", "Tank"]);
        $sa->random(2);
        // probably output: ["Neo", "Tank"]
        // probably output: ["Morpheus", "Tank"]
        // probably output: ["Trinity", "Tank"]
