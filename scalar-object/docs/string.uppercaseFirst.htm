uppercaseFirst

uppercaseFirst( ): string

Context: string.
Alias of ucfirst.
Make first character uppercase.

Returns a string with the first character of $scalarObject->val capitalized, if that character is alphabetic.

Note that 'alphabetic' is determined by the current locale. For instance, in the default "C" locale characters such as umlaut-a (Ã¤) will not be converted.

Return Values

The resulting string.
