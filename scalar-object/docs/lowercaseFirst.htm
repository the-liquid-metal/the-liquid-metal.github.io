lowercaseFirst

Returns a string with the first character of str , lowercased if that character is alphabetic.

Note that 'alphabetic' is determined by the current locale. For instance, in the default "C" locale characters such as umlaut-a (Ã¤) will not be converted.

Parameters

str

The input string.

Return Values

Returns the resulting string.

Example #1 lowercaseFirst() example

<?php
$foo = 'HelloWorld';
$foo = lowercaseFirst($foo);             // helloWorld

$bar = 'HELLO WORLD!';
$bar = lowercaseFirst($bar);             // hELLO WORLD!
$bar = lowercaseFirst(strtoupper($bar)); // hELLO WORLD!
?>
