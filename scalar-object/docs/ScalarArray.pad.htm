pad

pad (int $size, mixed $value): static

Pads to the specified length with a value.

see array_pad.

pad() returns a copy of the internal array padded to size specified by $size with value value. If $size is positive then the array is padded on the right, if it's negative then on the left. If the absolute value of $size is less than or equal to the length of the internal array then no padding takes place. It is possible to add at most 1048576 elements at a time.

Parameters

$size
New size of the array.
$value
Value to pad if internal array is less than $size.

Return Values

A copy of the internal array padded to size specified by $size with value value. If $size is positive then the array is padded on the right, if it's negative then on the left. If the absolute value of $size is less than or equal to the length of the internal array then no padding takes place. 

        $input = [12, 10, 9];

        pad($input, 5, 0);      // output: [12, 10, 9, 0, 0]
        pad($input, -7, -1);    // output: [-1, -1, -1, -1, 12, 10, 9]
        pad($input, 2, "noop"); // output: not padded
