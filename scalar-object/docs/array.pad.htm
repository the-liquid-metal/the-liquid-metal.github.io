pad

pad( int $size, mixed $value ): array

Context: array.
Alias of array_pad.
Pads to the specified length with a value.

pad() returns a copy of the $scalarObject->val padded to size specified by $size with value value. If $size is positive then the array is padded on the right, if it's negative then on the left. If the absolute value of $size is less than or equal to the length of the $scalarObject->val then no padding takes place. It is possible to add at most 1048576 elements at a time.

Parameters

$size
New size of the array.
$value
Value to pad if $scalarObject->val is less than $size.

Return Values

A copy of the $scalarObject->val padded to size specified by $size with value value. If $size is positive then the array is padded on the right, if it's negative then on the left. If the absolute value of $size is less than or equal to the length of the $scalarObject->val then no padding takes place.
