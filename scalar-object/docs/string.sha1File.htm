sha1File

sha1File( bool $rawOutput = false ): string

Context: string.
Alias of sha1_file.
Calculate the sha1 hash of a file.

Calculates the sha1 hash of the file specified by $scalarObject->val using the » US Secure Hash Algorithm 1, and returns that hash. The hash is a 40-character hexadecimal number.

Parameters

$rawOutput
When TRUE, returns the digest in raw binary format with a length of 20.

Return Values

A string on success, FALSE otherwise.
