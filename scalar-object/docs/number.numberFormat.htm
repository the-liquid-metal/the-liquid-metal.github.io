numberFormat

numberFormat( int $decimals, string $decPoint = ".", string $thousandsSep = "," ): string

context: number.
Alias of number_format.
Format a number with grouped thousands.

This function accepts either one or three parameters (not two):

If one parameters are given, $scalarObject->val will be formatted with $decimals decimals with a dot (".") in front, and a comma (",") between every group of thousands.

If all three parameters are given, $scalarObject->val will be formatted with $decimals decimals, $decPoint instead of a dot (".") before the decimals and $thousandsSep instead of a comma (",") between every group of thousands.

Parameters

$decimals
Sets the number of decimal points.
$decPoint
Sets the separator for the decimal point.
$thousandsSep
Sets the thousands separator.

Return Values

A formatted version of $scalarObject->val.
