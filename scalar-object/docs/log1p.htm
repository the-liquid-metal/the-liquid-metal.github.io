log1p( ): float

Context: number
Alias of log1p.
log(1 + number), computed in a way that is accurate even when the value of number is close to zero.

log1p

log1p() returns log(1 + $scalarObject->val) computed in a way that is accurate even when the value of $scalarObject->val is close to zero. log() might only return log(1) in this case due to lack of precision.

Return Values

log(1 + $scalarObject->val)
