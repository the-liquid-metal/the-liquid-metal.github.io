lastOccurSubstr

lastOccurSubstr( mixed $needle ): string

Context: string.
Alias of strrchr.
Find the last occurrence of a character.

This function returns the portion of $scalarObject->val which starts at the last occurrence of $needle and goes until the end of $scalarObject->val.

Parameters

$needle

If it contains more than one character, only the first is used. This behavior is different from that of firstOccurSubstr().

If it is not a string, it is converted to an integer and applied as the ordinal value of a character.

Return Values

This function returns the portion of string, or FALSE if $needle is not found.
