shift

shift (void): mixed

Shift an element off the beginning of internal array.

see array_shift.

Shifts the first value of the internal array off and returns it, shortening the array by one element and moving everything down. All numerical array keys will be modified to start counting from zero while literal keys won't be affected.

Return Values

The shifted value, or null if intenal array is empty.

        $arr = new ScalarArray("string", false, ["orange", 5 => "banana", 6 =>"apple", "rotten" => "raspberry", "peach"]);
        $fruit = $arr->shift();
        print_r($arr); // output: [0 => "banana", 1 => "apple", "rotten" => raspberry, 2 => "peach"]
