intdiv

Description

int intdiv ( int $numerator , int $divisor )

Returns the integer division of numerator by divisor.

Parameters

numerator

Number to be divided.

divisor

Number which divides the numerator.

Return Values

The integer division of numerator by divisor.

Errors/Exceptions

If divisor is 0, a DivisionByZeroError exception is thrown. If the numerator is PHP_INT_MIN and the divisor is -1, then an ArithmeticError exception is thrown.

Examples

Example #1 intdiv() example

<?php
var_dump(intdiv(3, 2));
var_dump(intdiv(-3, 2));
var_dump(intdiv(3, -2));
var_dump(intdiv(-3, -2));
var_dump(intdiv(PHP_INT_MAX, PHP_INT_MAX));
var_dump(intdiv(PHP_INT_MIN, PHP_INT_MIN));
var_dump(intdiv(PHP_INT_MIN, -1));
var_dump(intdiv(1, 0));
?>
